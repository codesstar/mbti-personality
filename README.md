# MBTI Personality for Claude Code

> 给你的 AI 编程搭档一个性格。

你有没有想过，Claude Code 不只是一个工具，而是你的**队友**？

这个 Skill 让你用一句话切换 Claude Code 的人格 —— 从「人狠话不多的技术大佬」到「脑洞大开的产品经理」，从 INTJ 冷面架构师到 ENFP 快乐小狗。它会改变 Claude 的思维方式、说话风格和做事节奏，但**不会**影响代码正确性和技术判断。

性格是调味料，不是主菜。

---

## 30 秒上手

**安装：**

```bash
# 在你的项目目录下
claude install-skill https://github.com/codesstar/mbti-personality
```

**使用：**

跟 Claude 说「切换人格」、「MBTI」、或者直接说一个类型名，就行了。

```
你：切换人格
你：用 INTJ 的风格
你：我要天才队友模式
你：换个性格
```

不需要记命令，自然语言触发。

## 4 个预设人格

每个预设都经过精心调配，覆盖最常见的编程场景。

### 1. 人狠话不多的技术大佬（INTJ × ISTP）

> 不解释，只甩 diff。你一看，全对。

- **思维**：全局规划 — 先在脑中构建系统终态，再倒推每一步
- **沟通**：惜字如金 — 能甩 diff 就不说话，代码本身就是解释
- **适合**：赶 deadline、性能优化、重构、脚本编写

### 2. 脑洞大开的产品经理（ENFP × INFJ）

> 「你真正需要的不是这个功能。」然后他说对了。

- **思维**：发散联想 — 灵感爆发式涌出方案，选最酷的那个
- **沟通**：洞察引导 — 穿透表面需求，挖掘真正的痛点
- **适合**：产品头脑风暴、快速原型、探索新方向

### 3. 带你飞的靠谱学长（ISTJ × ENFJ）

> 「来，先看文档怎么说的，我们一步步来。」

- **思维**：严谨务实 — 先查文档、先看先例，每一步都有证据支撑
- **沟通**：耐心引导 — 从简单到复杂逐步带着你走
- **适合**：写文档、code review、带新人、测试

### 4. 你的天才队友（ESTP × ENTP）

> 「别分析了，先写！」边写边怼，最后还真跑起来了。

- **思维**：行动优先 — 打开编辑器直接写，试错比分析快
- **沟通**：犀利挑战 — 质疑一切前提，给你新视角
- **适合**：MVP 开发、紧急修复、demo、hackathon

## 自定义组合

输入「自定义」，从三个维度自由搭配你的专属 AI 队友：

**思维方式** — 它怎么想问题？
- 全局规划 / 发散联想 / 严谨务实 / 行动优先

**说话风格** — 它怎么跟你聊？
- 惜字如金 / 热情洋溢 / 耐心引导 / 犀利挑战

**做事节奏** — 它怎么干活？
- 一步到位 / 快速迭代

4 × 4 × 2 = **32 种组合**，总有一款适合你。

## 智能推荐

不知道选哪个？输入「推荐」，告诉 Claude 你的 MBTI，它会根据**认知功能互补原则**帮你匹配最合拍的队友风格。

比如你是 ENFP（快乐小狗），创意满分但容易发散 → 推荐「人狠话不多的技术大佬」，帮你把天马行空收束成靠谱的代码。

不知道自己的 MBTI 也没关系 —— 说一下你在做什么，Claude 会按任务类型推荐。

## 工作原理

这个 Skill 通过 Claude Code 的 Skill 机制工作：

1. **即时模式**（默认）：人格仅在当前对话生效，不修改任何文件
2. **持久模式**（手动保存）：将人格指令写入 `CLAUDE.md` 的 `<!-- MBTI: -->` 块，跨会话保持

人格影响：沟通风格、思考方式、代码风格偏好、交互模式

人格不影响：代码正确性、安全性、工具使用、技术决策

## 灵感

MBTI 在编程圈不只是性格测试 —— 它描述了人解决问题的思维偏好。把这些偏好注入 AI，不是为了「准」，而是为了让人机协作更有**化学反应**。

一个 INTJ 风格的 Claude 会逼你先想清楚架构再动手。一个 ESTP 风格的 Claude 会拽着你先跑起来再说。这不是对错，是**互补**。

找到你最需要的那个队友。
